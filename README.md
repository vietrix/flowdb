# FlowDB

FlowDB là nền tảng quản trị cơ sở dữ liệu self-hosted, gồm backend Go và frontend web. Dự án tập trung vào trải nghiệm quản trị dữ liệu nội bộ với khả năng kiểm soát truy cập và vận hành linh hoạt.

## Mục tiêu

- Cung cấp workspace tập trung để kết nối và thao tác dữ liệu.
- Ưu tiên bảo mật, audit và vận hành ổn định.
- Dễ triển khai và mở rộng theo nhu cầu đội ngũ.

## Thành phần

- Backend: Go API server.
- Frontend: Next.js UI.

## Hỗ trợ dữ liệu

- PostgreSQL
- MongoDB
