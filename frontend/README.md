# FlowDB Frontend

## Công nghệ

- Next.js (App Router)
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Cài đặt & chạy local

```sh
pnpm install
pnpm run dev
```

## Scripts

- `pnpm run dev` - chạy dev server
- `pnpm run build` - build production
- `pnpm run start` - chạy production server
- `pnpm run lint` - chạy ESLint
